# 🏥 Hospital Data Analysis Dashboard in Excel

## 📋 Project Overview

This project demonstrates my Excel data analysis skills using a hospital dataset. I cleaned the raw data, performed data analysis using Pivot Tables and Pivot Charts, and created an interactive dashboard to summarize hospital performance.

---

## 🛠️ Tasks Performed

- Cleaned the dataset by handling missing values and duplicate records.
- Corrected inconsistent city names and formatted the data.
- Converted the dataset into an Excel Table.
- Created Pivot Tables for business analysis.
- Created Pivot Charts for visualization.
- Designed a Hospital Analysis Dashboard with KPIs.

---

## 📊 KPIs Included

- 💰 Total Revenue
- 👥 Total Patients
- 🌆 Top City
- 🏥 Top Department
- 👨‍⚕️ Top Doctor

---

## 📈 Pivot Table Analysis

- Bill Amount by City
- Bill Amount by Department
- Patient Count by Department
- Patient Count by Admission Type
- Average Bill Amount by Doctor

---

## 🔍 Key Insights

- Hyderabad generated the highest hospital revenue.
- Cardiology was the highest revenue-generating department.
- Dr. Sharma contributed the highest billing.
- Regular and Emergency admissions were analyzed to understand patient trends.
- Average billing varied across doctors and departments.
